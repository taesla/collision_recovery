Collision Recovery System

Doosan Robot M0609 • ROS2 Humble

GitHub Main Project

충돌 감지

로봇 상태를 실시간 모니터링하여 SAFE_STOP(5) 상태 전환을 즉시 감지합니다.

자동 복구

6단계 복구 시퀀스를 통해 안전하게 로봇을 정상 상태로 복원합니다.

ROS2 통합

ROS2 Humble과 완벽하게 통합되어 기존 시스템에 쉽게 적용 가능합니다.

Quick Start 

# Clone repository
git clone https://github.com/taesla/collision_recovery.git

# Navigate to directory
cd collision_recovery

# Run collision recovery
python main.py
